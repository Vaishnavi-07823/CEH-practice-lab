## 🧠 Key Concepts Documented

- **Five Phases of Ethical Hacking:** Reconnaissance → Scanning → Gaining Access → Maintaining Access → Clearing Tracks
- **Footprinting techniques:** WHOIS, DNS lookup, Google dorking, social media OSINT
- **Scanning types:** TCP Connect, SYN Stealth, UDP, XMAS, NULL scans
- **Enumeration:** NetBIOS, SNMP, LDAP, NTP enumeration methods
- **Attack types:** DoS, DDoS, MitM, Phishing, SQL Injection, XSS

## 🎯 Why CEH?

The cybersecurity landscape demands professionals who **think like attackers to defend like experts**. CEH provides a globally recognized, vendor-neutral framework for understanding attack vectors — making it essential for roles in:
- Penetration Testing
- SOC Analysis
- Threat Intelligence
- Incident Response
